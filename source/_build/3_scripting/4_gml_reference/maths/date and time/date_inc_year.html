date_inc_year

Returns a datetime after a given number of years have been added.

Syntax:

date_inc_year(date, amount);


Argument Description
date The datetime to add to
amount The number of years (must be an integer) to add.


Returns: Real


Description

With this function you can increment a given datetime value by a specific number of years, and it will return the new datetime value.


Example:

mynewdatetime = date_inc_year(date_current_datetime(), 1000);

This would set mynewdatetime to the current date but with 1000 years added.
