palette_get


Description

This function can be used to retrieve the different colour values for each of the 64 palette indices. You supply the unique palette ID to retrieve the indices from and the function will return a 64 value array, where each array entry corresponds to the palette index 0 - 63. Each entry will hold a 32bit colour value (includes the alpha component) with the format AABBGGRR.


Syntax:

palette_get(palette_id);


Argument Description
palette_id The unique palette ID value to get the colour indices from.


Returns:

array (64 colour values from [0] - [63])


Example:

var _parray = palette_get(global.pLight);
for (var i = 0; i < 64; i++;)
    {
    _parray[0] = _parray[0] | $ff000000;
    }
palette_set(global.pLight, _parray);

The above code retrieves an array of palette indices for the palette ID assigned to the global variable. It then sets the alpha value for all the array indices before passing it back to set the palette again.
