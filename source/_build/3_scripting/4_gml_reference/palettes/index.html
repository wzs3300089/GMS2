Palette Functions


Below is a list of all the functions that can be used for editing Palette Resources (for more information on Palettes, please see here). Note that when using these functions we refer to the palette_id which is the named constant for the palette resource or the unique value returned when you create the palette using palette_add, and the palette_index which is a value from 0 to 63 pertaining to each of the different colours within a palette resource.

  • palette_exists
  • palette_get_name
  • palette_get
  • palette_get_colour
  • palette_get_alpha
  • palette_set
  • palette_set_colour
  • palette_set_alpha
  • palette_add
  • palette_delete
  • palette_assign
  • palette_duplicate
