palette_delete


Description

This function can be used to delete a palette created using palette_add from memory. This function must be called whenever you no longer require the given palette resource (for example on calling game_restart or room_restart or when the instance using the custom palette has been destroyed), otherwise you run the risk of getting a memory leak. This happens when you create a resource and then remove or over-write the reference to this resource (but not the resource itself), meaning that it is taking up memory but is no longer available to GameMaker Studio 2 to use.


Syntax:

palette_delete(palette_id);


Argument Description
palette_id The unique palette ID value of the palette to remove from memory.


Returns:

N/A


Example:

if palette_exists(global.pLight)
    {
    palette_delete(global.pLight);
    }

The above code checks to see if the given palette ID exists, and if it does then it is removed from memory.
