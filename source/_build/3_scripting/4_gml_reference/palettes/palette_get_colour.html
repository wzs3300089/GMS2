palette_get_colour


Description

This function can be used to retrieve a colour value for any of the 64 palette indices of a given palette resource. You supply the unique palette ID to retrieve the indices from as well as the index of the palette to use (from 0 to 63), and the function will return an integer colour value (from 0 to 16777215) without the alpha component. If you need the alpha value use palette_get_alpha, or if you need the colour and alpha value use palette_get.


Syntax:

palette_get_colour(palette_id, palette_index);


Argument Description
palette_id The unique palette ID value to use.
palette_index The palette index to get the colour value from (0 - 63).


Returns:

Real (colour value from 0 to 16777215)


Example:

for (var i= 0; i < 64; i++;)
    {
    var _c = palette_get_colour(global.pLight, i);
    var _red = colour_get_red(_c);
    var _green = colour_get_green(_c);
    var _blue = colour_get_blue(_c);
    if ++_red > 255 _red = 0;
    var _new_c = make_colour_rgb(_red, _green, _blue);
    palette_set_colour(global.pLight, i, _new_c);
    }

The above code retrieves the colour component for each of the indices of the palette with the ID stored in the global variable. It then splits the colour value into its constituent parts and shifts the red value by one before creating a new colour value and using that to set the indices (looping the red value of the colour from 0 to 255).
