palette_set_alpha


Description

This function can be used to set the alpha value for any of the 64 palette indices of a given palette resource. You supply the unique palette ID to set the indices of as well as the index in the palette to use (from 0 to 63), and finally the alpha value (from 0 to 255). If you need to set the colour value use palette_set_colour, or if you need to set the colour and alpha value use palette_set).


Syntax:

palette_set_alpha(palette_id, palette_index, alpha);


Argument Description
palette_id The unique palette ID value to use.
palette_index The palette index to get the alpha value from (0 - 63).
alpha The alpha to use (from 0 to 255).


Returns:

N/A


Example:

for (var i= 0; i < 64; i++;)
    {
    var _a = palette_get_alpha(global.pLight, i);
    if_a - 1 > 0 _a--;
    palette_set_alpha(global.pLight, i);
    }

The above code retrieves the alpha component for each of the indices of the palette with the ID stored in the global variable. It then subtracts 1 from the alpha value if the value is greater than 0 before setting the palette index again using the new alpha value.
